# Crowd Mixer 2.0 – Community Edition

Crowd Mixer 2.0 – Community Edition (CMX CE) is a **free, streamlined edition** of Crowd Mixer 2.0, a Blender add-on for generating and managing crowds directly inside Blender using **Geometry Nodes**. It's built to help more Blender artists try crowd simulation at no cost.

This repository is the **public bug tracker** for CMX CE. It does not host the add-on's source code.

## What's included (free)

- Core crowd generation tools: **Standing**, **Flow Curve**, and **Stadium** crowd types
- - Basic preset saving and loading
 
  - ## Community Edition limits
 
  - - Some advanced features available in the full Pro version are not included in this edition
    - - The number of crowd characters/agents per scene is limited
     
      - For the full feature set and higher crowd limits, see full **Crowd Mixer 2.0 (Pro)**, available on [Superhive Market](https://superhivemarket.com/products/crowd-mixer) and [Gumroad](https://navalab.gumroad.com/l/CrowdMixer).
     
      - ## Installation
     
      - Install directly from the official Blender Extensions Platform:
     
      - **[extensions.blender.org/add-ons/crowdmixer-ce](https://extensions.blender.org/add-ons/crowdmixer-ce/)**
     
      - Installing this way lets Blender manage updates automatically. Requires **Blender 4.3 or newer**.
